# Shopify Middleware Dashboard
- Insights modules for RTO/In-Transit/OFD, TAT (Out of TAT, TAT Breach), Freight Discrepancies.
- Uses mock data by default. Flip `MOCK_MODE=false` + set env vars for live APIs.
- All times shown in IST.

## Quick Start
```bash
npm i
npm run dev
```
Open http://localhost:3000 and visit `/insights`.
